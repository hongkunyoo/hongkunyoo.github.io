Bom Project

Project BOM(Blindness Zero Movement) is an international development assistance launched by Yeonsei Severance Hospital(Dept. of ophthalmology) The project is held in lilongwe Malawi, Arifa. The main purpose is to prevent blindness for the local residents. I join the project as IT support faculty. My major work was to develop an medical record system with local NGO called 'Baobab'. The main part of mine was to develop an electronic eye chart with Ruby on Rails and OpenMRS. See more details here.

    • Develop period : 2013.07 ~ 2013.12
    • Service : Medical Record System
    • Language : Ruby on Rails / Ruby

At Here

2014 Microsoft Imagine Cup Innovation
Korea 1st Prize, World Semi Finalist

'At Here' brought a new paradigm of file sharing system.
'At Here' is a simple instant file sharing service. The differences between other services are, 'At Here' doesn't need any accounts. It uses location to recognize other users. You only need to be with them to share information.
