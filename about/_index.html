Phone Guitar Tab


Phone Guitar Tab is a guitar/bass/drums tab application for Windows Phone that provides search, download, upload, music streaming and many other capabilities to musicians while they are practising their favourite songs with their instruments.


  • #1 productivity tool for musicians on Store and very often featured by Microsoft.

  • Actively developed by 2 devs; release a month.

  • Open-source, ad-free and has much from the folks all around the globe.

PhoneCore: a boilerplate framework for WP apps.


PhoneCore is a framework that helps you build applications on Windows Phone. It provides:


  • MVVM framework features - ICommand, TriggerAction, INotifyPropertyChanged implementations which provide the way to react on user input.

  • Navigation engine that simplifies navigation between pages.

  • Dependency injection container which helps you create loosely coupled applications.

  • Aspect-oriented programming support by custom method interceptors using custom proxy class generated at compile time.

  • Bootstrapping engine with plug-in architecture.
